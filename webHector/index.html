Etiqueta Order

La propiedad "Order: ;" en CSS establece el orden en que se debe colocar un elemento flex o grid. Los elementos
de un contenedor se ordenan por valor de orden ascendente y luego por orden de código fuente. A los elementos que no tienen un valor de orden explícito se les asigna el valor predeterminado de 0.

Valor numérico incluyendo números negativos: order: 5; order: -5; Valores Globales: order: inherit; order: initial; order: unset;

Etiqueta Align-self

La propiedad align-self de Flexbox nos permite sobrescribir la alineación de un elemento dentro de
un contenedor flex, sea la alineación por defecto del navegador o la alineación especificada para los elementos en la propiedad align-items.

Sintaxis CSS

align-self: auto|stretch|center|flex-start|flex-end|baseline;
